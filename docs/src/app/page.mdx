# Coolify MCP

The Coolify MCP server helps agents interact with Coolify instances using a toolset focused on diagnostics, deployments, projects, and infrastructure.

## What you can do

- Manage apps, services, databases, and deployments
- Run diagnostics and inspect cross-instance resources
- Work with projects, teams, servers, and tokens
